## 注意事項
リポバッテリーを不適切に取り扱うと発火等の危険性があります。  
接続時の極性の向きや、リード線の短絡、組み立てには十分ご注意下さい。  
またPCBやケースの角、ピンヘッダーなど鋭利な箇所が多数存在していますので、怪我しないようご注意下さい。  
マイコン裏のパッドはバッテリーに接続されてるため、短絡させないようご注意下さい。  
取り扱いの不備により発生した怪我、事故等の責任は負いかねます。
